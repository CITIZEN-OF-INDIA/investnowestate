This project has been prepared as a static cPanel upload.

Use the ready-made package here:

- `cpanel-package/investnow-estate-cpanel.zip`

Upload that zip to your cPanel `public_html` folder and extract it there.

Notes:
- The zip already contains `index.html`, `assets/`, images, and `.htaccess`.
- If your domain uses the root `public_html`, extract the zip directly inside `public_html`.
- If you want it in a subfolder, extract it into that subfolder instead.
- This package is static HTML/CSS/JS, so Node.js is not required on cPanel.
